# 🎓 Learning Management System (MERN Stack)

A full-stack Learning Management System built using the MERN stack. The platform allows instructors to create and manage courses while students can enroll, track progress, and learn through an intuitive interface.

---

## 🚀 Live Demo

Frontend:
https://learning-management-system-tawny-one.vercel.app

Backend API:
https://learning-management-system-backend-eso8.onrender.com

GitHub Repository:
https://github.com/boopathiraja26/Learning_Management_System

---

## ✨ Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Role-Based Authorization

### Student
- Browse Courses
- Course Details
- Enroll in Courses
- Wishlist
- Learning Progress
- Student Dashboard
- Profile Management

### Instructor
- Instructor Dashboard
- Create Course
- Edit Course
- Delete Course
- Add Lectures
- Upload Videos
- Manage Students

### Additional Features
- Reviews & Ratings
- Cloudinary Image Upload
- Razorpay Payment Integration
- Responsive UI
- Protected Routes
